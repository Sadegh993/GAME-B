Unity WebGL prototype for a study on player experience.

Play here: https://github.com/Sadegh993/GAME-B

Controls:

Gamepad:

Left Analog Stick/D-Pad: Move

A: Jump

B or X: Shoot

Y: Melee Keyboard

Arrow keys / WASD: Move

Space: Jump

Ctrl: Shoot

Alt: Melee
